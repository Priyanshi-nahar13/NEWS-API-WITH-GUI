# 🧠 AI News API GUI

A simple Python-based desktop application that fetches the latest news using NewsAPI and displays it in a clean GUI built with Tkinter. The app also provides smart summarization of news headlines.

## 🚀 Features

* 🔍 Search latest news by topic
* 📰 Display top headlines with source
* 🔗 Clickable links to full articles
* 🤖 Smart summary of news
* 🎯 Simple and user-friendly interface

## 🛠 Tech Stack

* Python
* Tkinter (GUI)
* Requests (API handling)

## 🔑 API Setup

Replace the API keys in `app1.py`, "app.py":

```python
NEWS_API_KEY = "YOUR_NEWS_API_KEY"
GEMINI_API_KEY = "YOUR_GEMINI_API_KEY"
```

## ▶️ Run the Project

```bash
python app.py
python app1.py
```
